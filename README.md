## 1本目「自由な物質場の正準量子化」
### 概要
これは, 4次元Minkowski時空における自由な物質場の正準量子化についてのノートである. 内容は, どんな場の理論の教科書にも一番最初に書いてあるような基本的なことだが, それでもわざわざ書くのには主に2つの目的があるからである: (1)人によりがちなLorentz群の表現の表記を自分の中で統一すること. (2)簡略化されがちなフェルミオンの量子化をDirac括弧を使って厳密に行うこと.
